# Vehicle Management App

A Java application developed as part of my Web Application Development studies to manage different types of vehicles using object-oriented programming principles.

## Overview

This project implements a vehicle management system capable of handling different vehicle categories through inheritance and interfaces. The application demonstrates the use of object-oriented design patterns and software development best practices.

## Features

- Vehicle registration and management
- Different vehicle categories
- Sale and rental operations
- Object-oriented class hierarchy
- Interface implementation
- Data organization through custom classes

## Technologies

- Java
- Eclipse IDE

## Concepts Applied

- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Encapsulation
- Interfaces
- Method Overriding
- Class Hierarchies

## Project Structure

The application includes:

- Base vehicle class
- New vehicles
- KM 0 vehicles
- Second-hand vehicles
- Sale operations
- Rental operations

## Purpose

The purpose of this project is to practice object-oriented programming concepts, including inheritance and interfaces, through the implementation of a vehicle management system.
